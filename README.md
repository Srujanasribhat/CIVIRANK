# Smart Slum Development Priority Index

This is a Data Science mini project that calculates development priority scores for slum areas.

## Features

- CSV upload
- Data cleaning
- Feature normalization
- Weighted priority index calculation
- Risk classification
- KMeans clustering
- Interactive dashboard
- Map visualization
- Smart recommendations
- Download report option

## Tech Stack

- Python
- Streamlit
- pandas
- numpy
- scikit-learn
- Plotly
